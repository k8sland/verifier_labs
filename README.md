# <img src="assets/lab.png" width="32" height="auto"/> K8sLand Labs

## Description

Scores K8sland level1 labs.

To self score your lab:

1. cd to the corresponding lab number
2. Deploy!

      ```shell
      kubectl apply -f k8s
      ```

3. Your instructor will notify you with your score
