# K8sLand Level1 Lab Validation

This repo contains labs validators for K8sland level1 labs.

To validate your lab:

1. cd to the corresponding lab number
2. Deploy!

      ```shell
      kubectl apply -f k8s
      ```

3. Your instructor will notify you with your results
